Установка в OSPanel:
1. Распакуйте архив в папку домена, не меняя название проекта.
2. Запустите OSPanel от имени администратора.
3. Выберите модули:
   HTTP -> Nginx 1.22
   PHP -> PHP-8.3
   MySQL -> MySQL-8.0
4. Перезапустите OSPanel и откройте http://nova.local
5. При первом запуске проект сам создаст базу nova_shop и таблицы (нужны права root в локальном MySQL).
6. Если автосоздание БД отключено, создайте базу nova_shop вручную и импортируйте schema.sql.
7. Данные админа: admin / admin
8. Настройки БД лежат в config/db.php
